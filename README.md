# National--university--result-
## NU Result Checker

This is a simple website to check National University (Bangladesh) exam results.

## Features
- Check result by Roll No, Registration No, and Exam Year
- Displays Name and Result
- Uses JSON as the database
- Works completely online with GitHub Pages

## How to use
1. Go to [Visit Site](https://yourusername.github.io/repository-name/)
2. Enter your Roll Number, Registration Number, and Exam Year
3. Click "Check Result" button
4. Result will be displayed below

## Files
- `index.html` : Main web page
- `script.js` : JavaScript code to fetch and display results
- `data.json` : Student data
- `style.css` : Styling for the website
